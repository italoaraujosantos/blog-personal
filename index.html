Post #01: Motivos para aprender Python


  1. Profissionalmente emplementar infraestrutura como código;
  2. Desenvolvimento de métricas customizadas;
  3. Análise de Dados diversos;
  4. Análise quantitativos de ativos mobiliarios - Quant

O principal motivo de buscar aprender liguangem de programação já tão difundida em vários campos e diversas aplicabilidades dar - se pela nescessidade de emplementar infraestrutura como código, desenvolvimento de métricas customizadas que ajudem a melhorar a visão de negócio. Rotinas que visão desde a automatização de criar, manter ou destruir infra, uma vez que assim como deve ser facilmente criados os recursos os mesmos só são considerados bons se puderem ser destruídos e tão brevemente recriados no menor tempo possivel.

Análise de Dados esta em todo campo de conhecimento acadêmico, financeiro, riscos, reconhecimento facial e tantos outros. Não posso deixar no limbo esse conhecimento. Deve ser explorado até para melhorar a proposição de soluções como Engenheiro de Confiabilidade de Site.

Quant - Análise voltada para ativos mobiliarios. Assunto num futuro proximo espero explorar mais não visão teorica e ainda mais práticas.
